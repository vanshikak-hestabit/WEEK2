# Norway Blog: Seasons and Food

This ia a simple blog about Norway's seasons and food. It includes four seasons of Norway Spring, Autumn, Summer and Winter and popular dishes .

# How to use

-> open blog.html in your browser
-> you can click on the navigation links to directly jump on whichever topic you want to read

# Features

-> semantic HTML structure, no CSS
-> blog sections for seasons and food
-> contact info with name, email, message and submit button

# Learning

-> HTML fundamentals, semantics, page structure
-> ARIA labels 
-> built a simple page using only HTML
